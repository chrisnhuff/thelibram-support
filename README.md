# The Libram — Support & Issue Tracker

This repository is the public issue tracker for **The Libram**, an iOS character sheet app for tabletop RPGs, developed by Almost Right Creative LLC.

## Reporting an Issue

Before filing a new issue, please search the existing issues to see if your problem or suggestion has already been reported.

### Bug Reports

When reporting a bug, please include:

- **What happened** — A clear description of the problem.
- **What you expected** — What you thought should happen instead.
- **Steps to reproduce** — How can we make the bug happen again?
- **Device and iOS version** — e.g., iPhone 15, iOS 18.2
- **App version** — Found in the app's settings or in TestFlight.
- **Screenshots** — If applicable, include screenshots to help illustrate the issue.

### Feature Requests

We welcome ideas for improving The Libram. When suggesting a feature, please describe the problem you're trying to solve and how you'd like it to work. Context about your use case helps us prioritize.

### General Questions

For general questions about the app, feel free to open an issue with the **question** label.

## What to Expect

We review issues regularly. Please be patient — this is a small team. We may ask follow-up questions to better understand your report.

## Code of Conduct

Please be respectful and constructive in all interactions. Issues that are abusive or off-topic may be closed without response.
